👤 管理员 / 超级管理员

📊 数据概览

登录后首页，展示总订单数、各状态下单数、业务员人数、业务员订单排行。

📋 录新单

  1. 点击「订单新增」标签
  2. 填写到款单号（必填）→ 搜索选择业务员（支持模糊搜索）
  3. 添加产品明细：型号 / 数量 / 货期，可点「添加产品」增加多行
  4. 如需拍照，勾选「拍照请求」并填写拍照要求
  5. 点击「提交订单」，订单出现在操作员面板待处理区

📋 订单列表

数据概览页下方展示全部订单，表格包含：到款单号 · 单据日期 · 产品明细 · 总数量 · 状态 · 业务员 · 预计发货时间 · 拍照 · 拍照要求 · 带来 · 验货 · 收回 · 业务发货时间 · 创建时间 · 操作（删除）。

👥 业务员管理

  1. 点击「业务员管理」标签，查看全部业务员列表
  2. 添加业务员：姓名 + 识别码（组合唯一）
  3. 设置「归属主管」后，该业务员订单出现在主管的团队面板中
  4. 可停用/启用业务员

⚡ 超管专属

  1. 「⚙ 控制台」：管理用户账号
  2. 「🔧 操作员」：一键切换到操作员操作台
  3. 「👤 业务员」：一键切换到业务员面板（自动跳过识别码），查看全部业务员订单

🏭 操作员

🔍 智能筛选栏（7种筛选）

筛选触发条件使用场景
需拍照业务员勾选了拍照请求筛出需要拍照的订单
需带来业务员选了「是否带来：是」确认哪些单要拿过去
📅 待发货已到货 + 有验货时间/发货时间 + 未送货每天一筛，知道今天要送哪些
📅 待收回已到货 + 有收回时间 + 未收回验货/发货后筛出要拿回来的
📸 待拍照有拍照请求但未上传照片优先处理
📷 已拍照有拍照请求且已上传检查是否完善
📋 待完结有业务发货时间 + 未完结业务员填了发货时间后一键完结

📊 表格列顺序

到款单号 → 单据日期(可排序) → 产品明细 → 总数量 → 业务员 → 预计发货时间 → 进度回复 → 拍照 → 拍照要求 → 是否带来 → 验货时间 → 收回时间 → 业务发货时间(可排序) → 配送 → 操作

📦 完整操作流程

管理员录单 → 操作员改「已到货」→ 业务员选验货时间/是否带来 ↓ 操作员筛「待发货」→ 点【送货】→ ✓已送（可点击撤回） ↓ 业务员填「业务发货时间」→ 粉底高亮 ↓ 操作员筛「待完结」→ 改状态为「已完结」 ↓ （如需验货收回）业务员填「收回时间」 ↓ 操作员筛「待收回」→ 点【收回】→ 验货日期、是否带来自动清空 ↓ 支持多次循环：验货→送货→完结→收回→再验货→...

💡 快捷操作

  1. 状态下拉、预计发货时间、进度回复：直接编辑，自动保存
  2. 单据日期表头点击可排序（↑升序 / ↓降序）
  3. 业务发货时间表头点击可排序
  4. 已送出的订单，点 ↩️ 可撤销（退回待处理）
  5. 已收回的订单，清空验货/带来后支持重新送货
  6. 文件面板：点击操作列的附件按钮可上传/查看照片
  7. 刷新计时器：标题栏显示距上次刷新的时间

💼 业务员

🔐 身份确认

登录后输入你的识别码，系统自动识别身份。主管会额外显示「我的订单 | 👑 团队订单」切换栏。

🔍 查订单

打开页面自动展示你的全部订单。可按到款单号搜索，或按状态下拉筛选（选完自动搜，无需点搜索按钮）。「返回首页」可查看全部。

📝 订单操作字段

字段说明可用条件
📸 拍照请求需要操作员拍照 → 勾选并填写拍照要求（如"拍外箱+内件+序列号"）随时
✅ 是否带来需要操作员拿货过来 → 选「是」状态为「已到货」
📅 验货时间预计验货日期状态为「已到货」
📅 业务发货时间预计哪天发货（到货后才能选，选后粉底高亮）状态为「已到货」
📥 收回时间预计哪天操作员来收回订单已完结后
· 需要拍照 → 勾选「拍照请求」→ 填写拍照要求 → 操作员筛「需拍照」拍照上传 · 需要拿货 → 状态变已到货后 → 选「是否带来：是」+ 填验货时间 · 需要发货 → 状态变已到货后 → 填「业务发货时间」→ 操作员筛「待完结」改完结 · 需要收回 → 订单完结后 → 填「收回时间」→ 操作员筛「待收回」点收回

📝 进度回复

操作员填写的进度回复显示在订单卡片上，超20字自动截断。鼠标悬停可查看完整内容。

🔔 通知中心

  1. 右上角铃铛显示未读通知数（红底白字）
  2. 操作员修改订单状态、添加进度回复、上传照片后自动推送通知
  3. 点通知 → 自动搜索并定位到对应订单 → 该通知标为已读
  4. 「全部已读」：一键清空未读红点
  5. 「清空」：删除该业务员全部通知记录
  6. 每30秒自动检查新通知

👑 业务主管

  1. 登录后输入识别码，自动出现「我的订单 | 👑 团队订单」切换栏
  2. 团队订单显示下属每个业务员的名字及订单状态
  3. 可点击「⚠️ 逾期未收回」筛选超期订单（收回日期超过设定天数）
  4. 主管只读查看，不修改下属订单
