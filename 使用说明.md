# 进度回复系统 · 使用说明

## 👤 三种角色

| 角色 | 职责 |
|------|------|
| **管理员** | 创建订单、管理业务员、查看全部数据 |
| **仓库人员** | 更新订单状态、填写发货时间、上传文件 |
| **业务员** | 查看自己的订单、确认拍照/带来/验货时间 |

---

## 🔄 订单状态说明

```
待处理 → 调货中 → 路途中 → 已到货 → 已完结
```

状态由仓库人员按实际情况更新。

---

## ⚙️ 各角色操作指南

### 管理员

1. 在「业务员管理」添加业务员（设置姓名和识别码）
2. 在「订单新增」创建订单，填写到款单号、产品明细、所属业务员
3. 在「全部订单」可筛选、导出 CSV、清理旧文件
4. «数据概览» 查看各业务员订单统计

### 仓库人员

1. 在仓库操作台查看所有订单
2. 按实际情况更新订单状态
3. **改「已到货」前必须先填预计发货时间**
4. 可在「进度回复」栏填写备注
5. 点击文件按钮上传照片、单据

### 业务员

1. 登录后输入管理员分配的**识别码**确认身份
2. 只能看到**自己名下**的订单
3. 到款单号搜索或按状态筛选
4. 订单到货后可操作：

| 操作 | 何时可用 |
|------|----------|
| 拍照申请 | 订单有状态即可 |
| 是否带来 | 仅「已到货」状态 |
| 验货时间 | 仅「已到货」状态 |
| 收回时间 | 仅「已到货」状态 |

---

## ⚠️ 重要规则

- **「已到货」必须先填发货时间**才能选择
- **「是否带来」和「验货/收回时间」**只在已到货后可选
- **订单完结后**不可再修改拍照、带来、验货等信息
- **业务员只能看到自己的订单**，看不到其他人的数据
- 所有操作**实时生效**，刷新页面即可看到最新数据

---

## 📎 文件管理

- 仓库人员可为订单上传照片、单据
- 业务员可查看和下载文件
- 管理员可清理 N 天前的旧文件

---

## 📥 数据导出

管理员在订单列表点击「导出 CSV」即可下载全部订单数据。
